구글스토어에서 앱이 검색되지 않나요?

모두가 앞다투어
Web 3.0을 이야기 하는 시대

하지만,
유틸리티 없는 가짜 Web 3.0 프로젝트에 지친 유저들

유저가 받을 수 있는 인센티브가 부족한 현재의 Web 3.0 프로젝트들

그리고,
유틸리티는 있지만 태생적 한계를 지닌 Web 2.0 서비스들

글로벌 멤버십 서비스를 제공하기엔 비효율적이고 한계가 분명한 Web 2.0 시스템

WEB 2.0의 실생활 인프라와
WEB 3.0의 무한한 가능성을
연결할 순 없을까?

Web 3.0 시대의 문을 여는 FLEX한 멤버십의 시작

국경에 상관없이 글로벌에서 사용이 가능하며
더 많은 혜택을 멤버들에게 공정하게 제공해요

SERVICE

One Log in, All Membership

  • Web3 지갑 제공

    Web3Auth를 이용한
    지갑 서비스 제공

  • 글로벌 로그인

    한 번의 로그인으로
    전 세계 어디서나 즐기는 편리한 멤버십

  • 압도적 멤버십

    커넥트 파트너와 함께 제공하는
    라이프스타일 멤버십 베네핏

  • 유저 인센티브

    생태계 확장에 따른 유저 인센티브 증가
    (세이빙 프로그램)

MEMBERSHIP

당신의 라이프스타일에 딱 맞는 멤버십을 골라보세요
커넥트의 멤버십은 크게 3가지로 분류할 수 있어요

FLEX TYPE

똑같은 일상을 특별함으로 바꾸는 타입, 23개의 브랜드로 여유로운 가끔의 사치, 호캉스부터
나에게 주는 선물 같은, 뷰티&스파까지 FLEX하는 선물같은 혜택만 사용해보세요

SUBSCRIPTION TYPE

커넥트 플렉스 타입은 2가지 유형으로 가입할 수 있어요

  • 1.

    구독유형선택

    주간 미션 수행시마다 KCT를 추가 증정해요

  • 2.

    구독유형선택

    주간 미션 수행시마다 KCT를 추가 증정해요

  • 3.

    구독유형선택

    주간 미션 수행시마다 KCT를 추가 증정해요

INFRA

면세 유통업/인바운드 여행업 분야 1,3위 면세 유통업/인바운드 여행업 분야 1,3위 회사가
연매출 6조원의 인프라를 통해 혜택을 제공해요

  • 커넥트 유통 매출 총액

  • 파트너사

  • B2B 바이어

  • CHAINLINK BUILD

  • LISTED ON HUOBI

ECOSYSTEM

NFT

커넥트의 다양한 NFT로 재밌고 풍성한 혜택을 만나요

  • MEMBERSHIP NFT

    커넥트 멤버십 사용자에게 지급되는 NFT로
    카드를 합성하면 더 많은 혜택을 보유할 수 있어요

  • PFP NFT

    PFP NFT(Profile Picture)는
    ’커넥트 월드’의 캐릭터로 제작하였어요 프로필이나 증명서
    용도로 사용할 수 있으며 포함 혜택은 오픈 예정이에요

  • MYSTERY BOX NFT

    한정판으로 발매되는 랜덤박스 NFT예요
    커넥트 멤버십에서 제공하는 바우처를 통해
    할인가로 구매할 수 있어요

커넥트의 기존 NFT 마켓 플레이스를 방문하여 둘러보세요.
